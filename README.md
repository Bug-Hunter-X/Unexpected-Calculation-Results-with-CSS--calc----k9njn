# Unexpected Calculation Results with CSS `calc()`

This repository demonstrates an uncommon bug related to the CSS `calc()` function.  Specifically, the issue arises when combining percentages and other units (like pixels) within the calculation. This can lead to inconsistent rendering across different browsers.

The `bug.css` file contains the problematic CSS code. The `bugSolution.css` file offers a potential solution or workaround for this issue.  The README provides a detailed explanation of the problem and its solution.
